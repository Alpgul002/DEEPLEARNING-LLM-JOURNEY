# Machine Learning and Deep Learning Research Journey

This repository contains my academic work, homework assignments, lab notebooks, and research project materials related to **Machine Learning, Deep Learning, Convolutional Neural Networks, Transformer Architecture, and Artificial Intelligence**.

The repository represents my learning and research journey in modern AI systems. It includes both theoretical foundations and practical implementations, starting from convolution and neural networks and moving toward CNNs, Transformers, contrastive learning, self-supervised learning, fine-tuning large language models, and music genre classification.

### ML-HOMEWORK: Convolution and CNNs

Main topics:

* Music genre classification
* Audio feature analysis

* Logistic Regression
* PCA
* Support Vector Machine
* CNNs
* Log-Mel spectrograms
* Model comparison
* Supervised learning for audio classification
